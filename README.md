# 👶 Birth Weight Predictor (ML + Flask Web App)

A machine learning web application that predicts a baby's birth weight based on maternal and pregnancy-related features. Built using **Flask** and **Scikit-learn**, this project demonstrates end-to-end ML deployment from training to web integration.

---

## 🚀 Demo

Users can enter details like gestation period, age, height, weight, parity, and smoking status to get real-time predictions of baby birth weight.

---

## 🧠 Machine Learning Pipeline

- Data Preprocessing
- Feature Selection
- Model Training (Regression)
- Model Evaluation
- Model Serialization using Pickle (`.pkl`)
- Deployment using Flask

---

## 📊 Input Features

- Gestation
- Parity
- Age
- Height
- Weight
- Smoke (0 = No, 1 = Yes)

---

## 🎯 Output

- Predicted Baby Birth Weight
